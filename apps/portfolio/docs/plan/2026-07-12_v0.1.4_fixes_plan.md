## 🛠️ Proposed Changes (Implementation Details)

### 1. 커서 동작 개선 (Custom Cursor & Footer)

- **Footer.tsx**: 마지막 'SEUNGYEUB' 텍스트 영역의 `cursor-default` 클래스를 `cursor-none`으로 변경하여 기존처럼 시스템 커서를 숨깁니다.
- **CustomCursor.tsx**: CSS 조건부 클래스(`hidden [@media(hover:hover)_and_(pointer:fine)]:flex`)를 사용하여 터치 기반 모바일/태블릿에서는 커스텀 커서 렌더링을 완전히 숨기고, 마우스가 연결된 상태에서만 노출되도록 처리합니다.

### 2. 3D Canvas 수정 및 최적화 (Mobile)

- **InteractiveCardCanvas.tsx**:
  - `touch-pan-y` 클래스를 `touch-none`으로 수정하여 사원증 드래그 시 브라우저 스크롤 겹침을 방지합니다.
  - 모바일 등에서의 렌더링 최적화를 위해 Canvas `dpr={[1, 2]}`를 `dpr={[1, 1.5]}` 정도로 낮춰 부하를 줄입니다.
  - `ambientLight` 및 `<Lightformer>`의 intensity 강도를 높여 사원증이 더 밝게 렌더링되도록 수정합니다.

### 3. 레이아웃 & 텍스트 인터랙션 (Common & Mobile)

- **WorksSection.tsx**: 모바일용 "See All Works" 플로팅 버튼의 여백을 줄이기 위해 `bottom-10`을 `bottom-4` 정도로 조정하여 좀 더 아래로 이동시킵니다.
- **Footer.tsx**: 'SEUNGYEUB' 텍스트에 적용된 `hover:text-white`를 제거하고, `useInView` 훅을 활용해 뷰포트에 도달했을 때 호버 없이도 서서히 밝아지게 수정합니다.

### 4. 스크롤 애니메이션 고정 (Common)

- **WorksSection.tsx / HeroSection.tsx**: `viewport={{ once: false }}` 로 설정된 `motion.div` 요소들을 모두 `once: true`로 변경하여 반복 재생을 방지합니다.
- **ScrollRevealText.tsx**: 스크롤 연동 방식(`useScroll`)을 1회성 트리거 방식(`useInView` 및 `staggerChildren`)으로 전면 재작성하여, 텍스트가 한 번 밝아지면 스크롤을 올려도 다시 흐려지지 않도록 로직을 교체합니다.

### 5. Hero Section Marquee 리디자인 (Design)

- **HeroSection.tsx**: 하단의 Marquee 띠 부분에 대한 리디자인 작업을 별도로 진행하여 시각적 개선 방안을 테스트하고 적용합니다.
