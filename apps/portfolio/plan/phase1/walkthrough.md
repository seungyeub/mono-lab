# Phase 1: 포트폴리오 초기 세팅 완료 🎉

`frontend-foundation` 템플릿을 기반으로 나만의 개발 공간인 **`mono-lab`** 과 첫 번째 작품인 **`apps/portfolio`** 셋업이 성공적으로 완료되었습니다!

## 📌 주요 작업 내역

1.  **템플릿 복제 및 이름 변경**
    - `mono-lab` 최상위 패키지 이름 변경 완료.
    - `apps/web` ➡️ `apps/portfolio` 로 이름 및 경로 변경.
2.  **모노레포 의존성(pnpm) 설치 완료**
    - `pnpm install`을 통해 모든 워크스페이스 패키지 연결.
3.  **포트폴리오 핵심 라이브러리 추가**
    - `framer-motion` (애니메이션)
    - `@studio-freight/lenis` (부드러운 스크롤)
    - `zustand` (전역 상태 관리)
    - `lucide-react`, `clsx`, `tailwind-merge` (UI 유틸)
4.  **디렉토리 뼈대 생성**
    - `apps/portfolio/src/` 내부에 `features/`, `store/`, `contents/` 빈 폴더 생성 완료.
